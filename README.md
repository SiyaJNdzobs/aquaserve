# AquaServe Water Services Platform

## Overview
AquaServe is a web-based municipal service platform focused on water-related services. It allows users to report water issues, track requests, and simulate water bill payments.

## Features
- Report water issues (leaks, burst pipes)
- Track service requests
- View account balances
- Simulate payments
- User login/register (session-based)

## Tech Stack
- Django (Backend)
- HTML, CSS (Frontend)

## Notes
This is a prototype system:
- No real database integration
- No real payments
- Simulated logic only

## How to Run
1. Clone repo
2. Create virtual environment
3. Install Django
4. Run server
